Åpne én eller flere filer i MarkText.

Hvis brukeren oppgir filstier eller filnavn, kjør:
```
marktext <fil1> <fil2> ...
```

Hvis ingen filer er oppgitt, åpne prosjektets `docs/decisions/`-mappe:
```
marktext docs/decisions/
```

Kjør kommandoen via Bash-verktøyet. Du trenger ikke bekrefte — bare kjør den.
