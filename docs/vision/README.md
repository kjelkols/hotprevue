# Visjon og strategi

Dette området inneholder overordnet strategisk tenkning om Hotprevue og det større bildet systemet inngår i.

## Hva dette er

- Langsiktige ideer og retning
- Overordnede prinsipper som ikke passer i spec
- Eksplorasjon av muligheter som ikke er besluttet

## Hva dette ikke er

- Krav (se `spec/`)
- Handlingsliste (se `TODO.md`)
- Tekniske beslutninger (se `decisions/`)

## Bruk

Disse dokumentene brukes som strategisk kompass når nye funksjoner diskuteres. Før en større endring eller ny funksjon vurderes, sjekk om den er i tråd med retningen beskrevet her. Dokumentene skal ikke siteres som krav, men som kontekst.

| `philosophy.md` | Grunnleggende prinsipper og verdier for systemet |
